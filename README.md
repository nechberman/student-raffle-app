# מערכת הגרלות תלמידים 🎡

אפליקציית ווב אחת-לאחת (Vanilla HTML/CSS/JS, ללא Build tools) לניהול תלמידים, שיוך כרטיסי הגרלה, והרצת הגרלות עם גלגל מזל מונפש וקונפטי.

## קבצים
- `index.html` — מבנה האפליקציה
- `style.css` — עיצוב מודרני, RTL, אנימציות
- `script.js` — לוגיקה: תלמידים, שיוך כרטיסים, localStorage, גלגל מזל, קונפטי

## הרצה מקומית
פתחו את `index.html` בדפדפן (כפול-קליק), או הריצו שרת סטטי פשוט:

```bash
npx serve .
# או
python3 -m http.server 8080
```

## שימוש
1. **ניהול תלמידים** — הוסיפו תלמידים ברשימה הכללית.
2. **שיוך כרטיסים** — בחרו תלמיד, קבעו כמות כרטיסים (סיכויי זכייה) לכל אחת מ-16 ההגרלות, לחצו "שמור שיוך".
3. **מסך ההגרלות** — לכל פרס יש כרטיס עם כפתור "הרץ הגרלה". הגלגל מסתובב לפי משקל הכרטיסים של כל משתתף, עוצר על הזוכה, ונפתח פופ-אפ ניצחון עם קונפטי. תוצאה שמורה אוטומטית; ניתן לבטל ולהגריל מחדש דרך "↺ בטל ובצע הגרלה מחדש".

הנתונים נשמרים אוטומטית ב-`localStorage` של הדפדפן (כפתור "שמור הכל" למעלה שומר גם באופן מפורש).
